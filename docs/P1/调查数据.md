在你的校园生活中，有没有一些事情你觉得其实可以通过同学之间的互助更好地解决？你能结合自己的经历举几个具体例子吗？

- **访谈记录-同学A**：有，而且我感觉这种需求还挺多的。最常见的一个就是学习资料和课程信息，比如某门课老师不建群，或者作业要求说得比较散，这时候如果有学过这门课的同学愿意分享经验，会省很多时间。还有就是临时性的生活帮助，比如帮忙取快递、代拿外卖、帮忙看一下自习室有没有位置。对我来说，考试周的时候还会有一种需求，就是想快速找到愿意一起复习或者一起做项目的人，现在这种事情基本靠熟人转发，效率不太高。
- **访谈记录-同学B**：有，我觉得生活类的会特别明显。比如有时候人在仙林，东西却落在教学楼或者宿舍楼，自己回去拿很麻烦，如果有人顺路帮忙带一下，会方便很多。还有拼单、拼车这种，其实大家都有需求，但现在信息分散在不同群里，不太容易碰上合适的人。再比如活动或者比赛临时缺人，我身边同学经常是到处问朋友转发，最后找人很被动。对我来说，这些事情不是不能解决，而是现在解决方式很零散，很看运气。
- **访谈1**：校园跑，你帮我跑，我帮你跑，然后还有考试，每人负责整理一个学科的重点内容，想不到了。

### 问题2
如果有一个面向校内同学的互助平台，你觉得它最应该具备哪些功能？为什么这些功能对你来说比较重要？

- **访谈记录-同学A**：我觉得第一是信息分类要清楚，比如学习求助、生活跑腿、二手交易、组队招募这些要分开，不然会很乱。第二是站内沟通功能要有，而且最好不要一上来就交换私人联系方式，因为很多人会有顾虑。第三是搜索和筛选要好用，比如按校区、时间、课程名称、需求类型去找。第四我觉得评价功能也很重要，不一定做得特别复杂，但至少要让我知道这个人以前靠不靠谱。对我来说，这几个功能重要，是因为我不想为了一个小需求在很多群里到处问，也不想承担太大的沟通成本。
- **访谈记录-同学B**：我比较看重三个方向。一个是发布需求要方便，最好几步就能发出去，而且模板比较清楚，不然很多人懒得写。一个是匹配功能，比如我发的是拼车或者找搭子，系统最好能推荐比较合适的人，而不是全靠别人刷到。还有一个是隐私保护，比如前期先在平台里交流，不要直接暴露电话或者微信。我觉得这些功能重要，是因为很多人不是没有需求，而是怕麻烦、怕尴尬、也怕遇到不靠谱的人。要是平台能把这些顾虑降下来，大家才会更愿意开口求助。
- **访谈1**：聊天，这肯定要有吧；心灵树洞，可以发帖，可以倾诉，受众广，可以吸引很多人，在评论区可以更加精准满足自己的需求。
- **访谈2**：应该具备一些解决大学生常见需求的功能，例如代取快递、代课、校园跑代跑……
- **同学1**：我觉得最该有需求发布 / 接单、校内实名认证、即时聊天这几个功能。需求发布能直接说自己要帮什么忙，不用到处问；实名认证是因为都是校内同学，靠谱不担心被骗；即时聊天就不用加微信，沟通起来快，省时间。

### 问题3
你觉得同学在使用这类平台时，通常最在意的是什么，比如效率、信任、安全感、便利性，还是别的因素？你自己的想法是什么？

- **访谈记录-同学A**：我觉得大部分人最在意的还是信任和便利性。因为如果只是方便但不安全，大家不敢用；如果很安全但流程特别麻烦，大家也懒得用。我自己的排序大概是信任第一，效率第二，便利性第三。信任包括身份真实、评价可查、出问题有人处理。效率主要是我发出需求后能不能很快有人响应，或者我能不能快速搜到自己想要的信息。像学生其实日常都挺忙的，尤其是到期中期末的时候，操作太复杂的平台很难留住人。
- **访谈记录-同学B**：我觉得我会先看安全感，然后才是效率。因为这类平台会涉及见面、交易，甚至可能有一点个人信息暴露，所以如果没有基本的安全保障，我不会轻易用。其他同学可能有人更看重效率，但我觉得最后还是会回到一个问题，就是能不能放心地跟对方互动。比如对方是不是校内同学、有没有信用记录、如果出了问题平台管不管。我自己的感受是，便利性可以慢慢优化，但安全感如果一开始做不好，平台口碑很容易直接坏掉。
- **访谈1**：隐私和安全感吧，大多数人是不愿意把自己的vulnerability展示给其他人的
- **访谈2**：我最在意平台的效率，也就是说我发布一个问题希望能匹配到解决我问题的人，也许可以在问题标签上下功夫，比如发布的标签精确到某种任务、某节课。
- **同学1**：我觉得大家最在意信任和安全，其次是效率。毕竟都是校内交易和帮忙，怕遇到不靠谱的人、信息泄露，或者被放鸽子。然后就是效率，大家都忙，不想等太久，能快速匹配到人最好。我自己也是，先看安不安全、靠不靠谱，再看方不方便。

### 问题4
从你的角度看，一个校园互助平台要真正被大家接受并长期使用，最关键的条件有哪些？

- **访谈记录-同学A**：我觉得第一个条件是要真的有用，不能只是看起来功能很多，但实际没人发、没人接。平台早期可能要先聚焦几个高频场景，比如课程资料、跑腿、失物招领，这些比较容易形成使用习惯。第二是要有基本的规则，比如什么能发、什么不能发，纠纷怎么处理，恶意放鸽子怎么办。第三是页面和流程得简单一点，学生不会愿意花很多时间学怎么用。还有一个我觉得容易被忽略，就是平台最好有一点激励，不一定非要现金，也可以是积分、信誉等级或者一些校内权益，不然很多互助行为最后可能只靠少数热心同学撑着。
- **访谈记录-同学B**：我觉得首先是氛围，不能让人觉得这里很乱或者很容易吵架。很多平台功能其实都差不多，但氛围差，大家就不会留下来。第二是规则要明确，比如什么类型的任务可以发、收费类怎么界定、放鸽子或者恶意交易怎么处理。第三是要持续有人维护，不能刚开始看起来很好，后面帖子过期了也没人清理。还有一点是它最好能形成一种习惯场景，比如大家一想到找搭子、找跑腿、找二手，就会自然想到这个平台，这样它才有可能长期活下去。
- **访谈1**：首先平台运行要稳定，再就是要有它的不可替代性，有些功能只能在这个平台上才能完成，或者能够给用户一种独一无二的感觉，其次就是前期的用户累计要做到全年龄段覆盖，最后也是最重要的是要能真正迎合用户的切实需求，比如考研、保研、参军这些非常实际东西以及婚恋等大家可能都比较感兴趣的问题，在这些方面上能够给出具体的建议与帮助。
- **访谈2**：我觉得得做好和校园集市的区分，毕竟我在集市上发个吐槽帖渴望获得他人的安慰也算一种精神帮助。
- **同学1**：想让大家长期用，首先必须全是本校学生，严格实名，杜绝校外人员，安全第一；然后操作要简单，别太复杂，打开就能用；最后氛围要好，没有乱收费、没有骗子，大家互帮互助，用着舒服才会一直用。

### 问题5
假如你已经开始使用这样的平台，在什么情况下你会愿意主动发布需求、帮助别人，或者把它推荐给身边同学？

- **访谈记录-同学A**：如果我发现它真的能解决问题，我会愿意持续用。比如我发课程相关的问题，真的有人认真回答；或者我临时有个跑腿需求，很快有人接，那我下次就还会用。帮助别人这件事我也愿意，但前提是成本别太高，比如顺路带个快递、分享一下自己上过的课程经验，这种我会做。如果平台的评价机制清楚、沟通也比较顺畅，我也会推荐给同学。反过来说，如果上面广告很多、信息乱、经常没人回应，或者有人态度很差，我应该很快就不会再用了。
- **访谈记录-同学B**：如果我第一次使用体验还不错，我就会继续用。比如我发布一个拼单需求，很快有人回应，过程也比较顺利，我会觉得这个平台值得留着。帮助别人这件事我也不是完全排斥，但我更倾向于自己顺手能做的，比如顺路带东西、回答一些自己了解的问题。推荐给朋友的话，前提是我自己先用得安心，尤其是不会出现骗子很多、信息失真或者莫名其妙被骚扰这种情况。要是平台能让我觉得省心，而且上面的人整体比较友善，我应该会愿意推荐。
- **访谈1**：自己通过其他渠道无法解决自身问题我会选择用它发布需求，可能每天定期看看有没有人想要寻求帮助，就像检查邮件一样；当我通过这个平台切实解决自身的难题或者能够给他人带来帮助，我会把它推荐给身边的同学，或者说身边同学有困难无法解决，我会向他推荐。
- **访谈2**：把2、3、4的问题做好
- **同学1**：我自己有急事解决不了的时候，比如急着取快递、找资料，就会发需求；看到别人的忙我顺手能帮，也会主动接单。推荐给同学的话，就是我自己用着觉得安全、靠谱、真的能解决问题，而且不麻烦，就会安利给室友和同班同学。
