# 调查问卷 + 访谈提纲

## 调查问卷

1. 给出你希望有的板块
2. 请问你认为互助平台应该包含哪些功能？
3. 以上功能或板块中，你认为最重要的是？
4. 你认为通过学号认证注册方式如何？
5. 为了让平台活下去且维持良好氛围，你认为需要管理哪些方面？
6. 你希望平台的收费方式是？
7. 如果平台需要志愿管理员，你的应聘意愿是？
8. 你对平台建设还有哪些建议？
9. 要是平台真建成了且真投入使用了，你的使用意愿是？
10. 你在生活中还有什么需要他人参与协助的需求？
11. 你在互助平台上想要得到怎样的酬劳，你愿意接受本平台的积分吗？
12. 在互助的过程中，你认为需要与对方进行线上交流吗？
13. 你认为平台应该通过什么标准来判断一个互助任务成功或者失败？

## 访谈提纲

| 序号 | 访谈问题 |
|---|---|
| 1 | 在你的校园生活中，有没有一些事情你觉得其实可以通过同学之间的互助更好地解决？你能结合自己的经历举几个具体例子吗？ |
| 2 | 如果有一个面向校内同学的互助平台，你觉得它最应该具备哪些功能？为什么这些功能对你来说比较重要？ |
| 3 | 你觉得同学在使用这类平台时，通常最在意的是什么，比如效率、信任、安全感、便利性，还是别的因素？你自己的想法是什么？ |
| 4 | 从你的角度看，一个校园互助平台要真正被大家接受并长期使用，最关键的条件有哪些？ |
| 5 | 假如你已经开始使用这样的平台，在什么情况下你会愿意主动发布需求、帮助别人，或者把它推荐给身边同学？ |
